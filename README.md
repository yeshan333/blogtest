# Flask Blog

> attack on Flask, support article search

## dependencies

- Flask
  - flask-login
  - flask-mail
  - flask-wtf
  - flask-moment
  - flask-sqlalchemy
  - bootstrap-flask
  - flask-whooshee
- Font Awesome
- Jquery
- Bootstrap
  - Bootswatch

## local preview

```bash
pip install pipenv
pipenv install  # install dependencies
pipenv shell  # activate virtual environment
flask run  # start server
```
